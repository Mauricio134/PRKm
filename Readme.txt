2. Explicacion para ejecutar el codigo:
- Compilar el código con la siguiente línea en la consola: g++ -std=c++14 main.cpp -o main.exe

Sin embargo, cuando ejecuté mi trabajo en windows, parece que mi proyecto suele consumir muy rápido la memoria RAM,
cosa que no ocurre en mac. No estoy seguro de como arreglarlo a estas instancias del trabajo, pero lo más recomendable es que
se ejecute este programa de una OS mac para que pueda correr sin problemas.
